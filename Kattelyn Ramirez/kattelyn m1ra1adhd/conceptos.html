Sitio Web
Un sitio web es un conjunto de páginas web relacionadas que están almacenadas en un servidor y accesibles a través de internet mediante una URL. Puede contener información, imágenes, videos, formularios y otros elementos interactivos. Ejemplo: Amazon, Wikipedia, YouTube.
Página Web
Es un documento digital escrito en HTML que forma parte de un sitio web. Puede ser estática (contenido fijo) o dinámica (contenido que cambia según la interacción del usuario).
Hipervinculo
También llamado enlace, es un elemento que permite saltar de una página a otra dentro de un sitio web o a otro sitio en internet. Se representa con la etiqueta en HTML.
URL (Uniform Resourse Locator)
Es la dirección única que identifica un recurso en internet. Está compuesta por el protocolo (https://), el dominio (www.ejemplo.com) y la ruta del recurso (/pagina).
HTML (HiperText Markup Lenguage)
HTML (HyperText Markup Language) Es el lenguaje de marcado que estructura el contenido de una página web mediante etiquetas. Define qué elementos aparecen y su orden en la página.
CSS (Cascading Style Sheets)
Es un lenguaje de estilos que se usa para diseñar páginas web, definiendo colores, fuentes, márgenes, tamaños y más.
JS (JavaScrip)
Lenguaje de Programación que añade interactividad a las páginas web
DNS (Domain Name Sistem)
Sistema que traduce nombres de dominio en direccciones IP.
IP
Dirección de numérica que identifica un dispositivo en una red
Dominio Web
Nombre único que identifica un sitio web en internet (Ej. Google.com)
Cliente-servidor
Modelo en el que un cliente (navegador) solicita información a un servidor
FTP (File Transfer Protocol)
Protocolo para transferir archivos entre computadoras en una red
HTPP (HiperText Transfer Protocol)
Protocolo que permite la comunicación en la web.
HTTPS (HTTP Secure)
Versión segura de HTTP que cifra los datos para mayor seguridad
Frontend
Parte visual e intereactiva de un sitio web.
Backend
Parte lógica de un sitio web que gestiona datos y funcionalidad en el servidor.
Hosting
Servicio que almacena ditios web y los hace accesibles en internet.
Hosting Privado Virtual (VPS)
Tipo de Hosting que ofrece recursos dedicados dentro de un servidor compartido
Hosting de la Nube (Cloud Hosting)
Alojamiento web distribuido en multiples servidores en la nube
W3C (World Wide Consortium)
Organización que establece estándares para el desarrollo web
